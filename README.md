# AI Problem Solving Assignment

## Problem 1: Drone Delivery System

### Description:
This program simulates a drone delivering packages using search algorithms in a grid-based environment.

### Algorithm Used:
Breadth First Search (BFS)

### How to Run:
python problem1/drone_delivery.py

### Output:
A GUI window opens showing the drone navigating and delivering packages.

## Problem 2: GPS Route Finder

### Description:
This program finds the shortest path between locations using an AI-based pathfinding algorithm.

### Algorithm Used:
A* Search Algorithm

### How to Run:
python problem2/gps_route_finder.py

### Output:
A GUI map displaying the optimal route between points.
